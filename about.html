About

This website was created as an assignment for the course '2ID60: Web Technology' taught at the Eindhoven University of Technology.

The goal of this website is to give users guidance on what to vote during the upcoming elections in the Netherlands.

Bootstrap was used to create the layout of the website. The collapsible menu on mobile phones was also created using Javascript code provided by Bootstrap.

JQuery (JavaScript framework) was used to handle user events. JQuery's accordion class was used on the elections.html page along with the styling of JQuery-ui.
